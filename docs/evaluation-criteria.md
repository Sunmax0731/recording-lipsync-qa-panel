# 評価基準

QCDS は Quality、Cost、Delivery、Satisfaction と定義します。

| 観点 | A- 以上の条件 |
| --- | --- |
| Quality | 代表シナリオ、required docs、security/privacy、traceability がそろう |
| Cost | 追加グローバルツールなしで検証でき、運用負荷が限定される |
| Delivery | GitHub prerelease、docs ZIP、release evidence がそろう |
| Satisfaction | 手動テスト手順が実施可能で、次アクションが分かる |
