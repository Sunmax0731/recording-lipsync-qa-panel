# 競合ベンチマーク

        ## 比較対象

        - Adobe Character Animator: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Premiere Pro markers: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Google Sheets: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。

        ## 採用する基準

        - 入力負荷が低い。
        - 次アクションが明確。
        - ローカルまたは閉域で検証できる。
        - docs と release evidence から状態を追える。

        ## 差別化

        Adobe Character Animatorの収録テイク、口パク違和感、再収録判断を記録するQAパネルに絞り、汎用ノートや大型サービスよりも閉域アルファの検証可能性を優先します。
