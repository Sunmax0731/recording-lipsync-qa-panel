# Security / Privacy Checklist

## 方針

音声や動画本体は扱わず、テイクIDと検品メモだけを保存する

## チェック

- [x] 外部送信をMVP外にした。
- [x] サンプルに実個人情報を含めない。
- [x] 手動テストで確認すべき権限を列挙。
- [x] release asset に手動テスト手順を添付。

## Android / Host 権限

- 追加のOS権限は閉域アルファMVPでは要求しない。
